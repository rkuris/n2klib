Detailed Description

Message assembler from Packet.

This class contains all the stuff needed to construct N2k::Message objects from Packets. It expects for someone to call Receiver::ingest on each received packet.

◆ ingest()

void n2k::Receiver::ingest ( const Packet &  packet)
inline

ingest inserts a packet into the appropriate Message in the MessagePool and calls the registered Callback. If there aren't any callbacks, this method does nothing.

Care is taken to call all the callbacks, allowing duplicates in the callback list.
