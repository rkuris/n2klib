Detailed Description

Contains a single NMEA 2000 message.

Message is a class containing a single NMEA message. These can be assembled from Packet classes, typically from the CAN bus, or transmitted by splitting them into a stream of Packet classes.

User's typically don't use this class directly, instead they use a Transciever (or Receiver) class which has a MessagePool used for assembling a Message from one or more Packet classes, and a derived class for fetching the fields of the Message by name. All the named NMEA2000 classes extend this.
